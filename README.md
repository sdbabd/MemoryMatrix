# 考研记忆 Pro 

## 📚 项目简介

考研记忆 Pro 是一个专门为考研学生设计的智能记忆系统，主要覆盖数字信号处理（DSP）的核心概念和公式。该系统采用基于遗忘曲线的智能遮盖技术，帮助用户高效记忆关键知识点和数学公式。

## 🎯 核心特性

### 🧠 智能记忆系统
- **自适应遮盖率**：可调节的难度设置（20%-90%遮盖率）
- **LaTeX公式支持**：完美渲染复杂数学公式
- **记忆热力图**：可视化显示知识点的掌握程度
- **专注模式**：点击遮盖区域进入专注答题模式
- **状态持久化**：答题状态在页面切换时完整保持（v1.1新增）

### 📊 学习数据分析
- **遗忘次数统计**：记录每个知识点的遗忘情况
- **错误率计算**：实时计算整体学习效果
- **平均耗时分析**：统计每个知识点的掌握时间
- **热力图可视化**：5级颜色编码显示掌握程度

### 🎨 现代化界面
- **响应式设计**：完美适配桌面和移动设备
- **Material Design**：现代化的卡片式界面
- **平滑动画**：流畅的过渡和交互效果
- **暗色模式支持**：保护眼睛的界面设计

## 📖 题目内容概览

### 1. 时域采样定理
**核心概念**：带限信号的等间隔采样原理
- 采样信号的频谱是原信号频谱的周期延拓
- 奈奎斯特条件：ωₛ ≥ 2ωₘₐₓ
- 理想低通滤波器的恢复作用

### 2. 变换之间的关系
**系统概念**：各种变换的相互联系
- 拉普拉斯变换(LT)与傅里叶变换(FT)
- Z变换与拉普拉斯变换的关系
- 离散时间傅里叶变换(DTFT)与Z变换
- 离散傅里叶变换(DFT)的采样特性

### 3. 拉氏变换与Z变换对应关系
**映射关系**：z = eˢᵀ 的物理意义
- s平面左半平面 → z平面单位圆内
- s平面右半平面 → z平面单位圆外
- s平面虚轴 → z平面单位圆上

### 4. 无失真传输
**条件要求**：y(t) = K·x(t-t₀)
- 波形完全相同，仅幅度和时延不同
- 系统函数的幅频特性和相频特性要求

### 5. 调制及其作用
**信号处理**：低频信号到高频段的搬移
- 提高传输效率
- 实现信号复用
- 减少传输失真
- 减小天线尺寸

### 6. 调幅系统比较
**AM、DSB、SSB特性对比**：
- AM：包含载波，功率效率低
- DSB：抑制载波，双边带传输
- SSB：单边带传输，带宽效率最高

### 7. 多路复用技术
**时分复用(TDM)与频分复用(FDM)**：
- TDM：基于采样定理的时域分隔
- FDM：基于频谱搬移的频域分隔

### 8. 全通系统特性
**零极点分布规律**：
- 连续系统：零极点关于虚轴对称
- 离散系统：零极点关于单位圆镜像对称
- 相频特性单调递减

### 9. DFT频谱分析现象
**四大现象及解决方案**：
- 频谱混叠：增加采样频率
- 频谱泄露：延长窗长度或加窗
- 栅栏效应：增加抽样点数或补零
- 谱间干扰：选择合适的窗函数

### 10. 吉布斯现象
**数学现象**：傅里叶级数的收敛特性
- 不连续点处的9%过冲
- 衰减振荡形式
- 不随项数增加而消失

### 11. 频域采样定理
**采样条件**：N ≥ M才能完全恢复
- 有限长序列的频域采样
- 时域恢复的条件和限制

### 12. IIR滤波器设计方法
**设计方法对比**：
- 脉冲响应不变法：时域逼近，存在混叠
- 双线性变换法：解决混叠，频率非线性

### 13. IIR与FIR滤波器区别
**结构特性对比**：
- IIR：有反馈，阶数低，非线性相位
- FIR：无反馈，易实现线性相位，一定稳定

### 14. IIR滤波器结构
**实现结构特点**：
- 级联型：便于调节零极点，速度较慢
- 并联型：速度快，误差小，只能调极点

### 15. 时域补零效果
**分辨率概念澄清**：
- 补零不提高物理分辨率
- 只改变DTFT的观察方式
- 消除栅栏效应的手段

### 16. 多采样率转换
**采样率转换问题**：
- 上采样：影像频谱现象
- 下采样：频谱混叠问题
- 抗混叠滤波器的应用

## 🚀 使用指南

### 基本操作
1. **开始学习**：打开应用即可开始第一题的学习
2. **记忆测试**：点击被遮盖的文字或公式
3. **答题判断**：选择"记得"或"忘了"
4. **切换题目**：使用底部的"上一题"/"下一题"按钮

### 高级功能
1. **热力图查看**：点击顶部火焰图标查看记忆热力图
2. **设置调整**：点击齿轮图标调整遮盖难度
3. **重置练习**：完成所有题目后可选择重新开始
4. **数据统计**：实时查看遗忘次数、错误率和平均耗时

### 热力图颜色说明
- 🟢 **0次**：完全掌握（绿色）
- 🟡 **1-2次**：基本掌握（黄色）
- 🟠 **3-4次**：需要练习（橙色）
- 🔴 **5-7次**：重点复习（红色）
- 🟣 **8次+**：难点重点（紫色）
